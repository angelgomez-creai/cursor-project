## Tecnologías Utilizadas

### Stack Principal
- **React 18**: Framework frontend moderno con hooks
- **TypeScript**: Tipado estático para mayor robustez
- **Vite**: Build tool ultra-rápido para desarrollo
- **React Router**: Navegación SPA moderna

### UI y Estilos
- **Ant Design**: Biblioteca de componentes empresarial
- **CSS Modules**: Estilos encapsulados y mantenibles
- **Responsive Design**: Adaptación a múltiples dispositivos

## Componentes Principales

### App.tsx
El componente raíz que define:
- Router principal con React Router
- Layout de la aplicación (Header, Content, Footer)
- Rutas básicas (expandible para módulos futuros)

```typescript
// Estructura básica del App component
<Router>
  <Layout>
    <AppHeader />
    <Content>
      <Routes>
        <Route path="/" element={<HomePage />} />
        {/* Rutas futuras para productos, carrito, etc. */}
      </Routes>
    </Content>
    <Footer />
  </Layout>
</Router>
```

### Header Component
- Navegación principal
- Branding de la aplicación
- Menu responsive (preparado para más secciones)

### HomePage
- Landing page del proyecto
- Componente de demostración
- Base para funcionalidades futuras

## Scripts de Desarrollo

### Configuración Inicial
```bash
# Setup automático de dependencias
./setup.sh    # Linux/Mac
setup.bat     # Windows
```

### Desarrollo
```bash
# Servidor de desarrollo
./start.sh    # Linux/Mac
start.bat     # Windows

# O directamente:
pnpm dev      # Inicia en localhost:5173
```

### Build y Deploy
```bash
pnpm build    # Build para producción
pnpm preview  # Preview del build
pnpm lint     # Linting del código
```
